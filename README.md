## Contributing

Contributions welcome! Please read the [contributing guidelines](CONTRIBUTING.md) first.

- Tools must be actively maintained (last release within 12 months)
- Include real pricing — check the vendor's pricing page, not your memory
- One PR per tool addition — keep diffs reviewable
- Descriptions should explain *who it's for*, not just *what it does*
