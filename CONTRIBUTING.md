# Contributing to Awesome Knowledge Base Tools

Thanks for helping keep this list accurate and useful.

## How to Add a Tool

1. **Fork** this repo and create a branch: `add-toolname`
2. **Add your entry** in the appropriate category, alphabetically or by relevance
3. **Format**: `| [Tool Name](URL) | One-sentence description — who it's for and what makes it distinct | Pricing |`
4. **Open a Pull Request** with the title: `Add ToolName to [Category]`

## Quality Standards

- **Actively maintained** — last update within 12 months
- **Real pricing** — check the vendor's pricing page; use "Contact for pricing" only if truly unavailable
- **Honest descriptions** — explain who the tool is for, not just what it does
- **No affiliate links** — plain links only

## What Not to Add

- Tools that have been abandoned or shut down
- Duplicate entries
- Personal note-taking apps that aren't team-focused (Obsidian, Roam, etc. belong in a different list)
- Your own product without disclosure

## Reporting Issues

Open an issue if:
- A tool has been discontinued or acquired
- Pricing is significantly outdated
- A description is misleading
